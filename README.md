# Coffee-Sales-Analysis
This project analyzes coffee sales data and presents insights through an interactive dashboard.
The goal is to demonstrate how a data analyst can transform raw sales transactions into actionable business recommendations.


Dimensions Analyzed:

Time of Day (Morning, Afternoon, Night)

Hour of Day (peak hours 10–12 AM, 3–7 PM)

Month (seasonality trends)

Coffee Type (Latte, Americano, Cappuccino, etc.)

Weekday patterns
